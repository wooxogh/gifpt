## Design System

Before writing any UI component, read `../DESIGN.md`.

- Aesthetic: **Cinematic Dark** — dark canvas, electric violet accent
- Colors: bg `#0a0a0f`, surface `#13131a`, accent `#7c6af7`
- Fonts: Geist (sans) + Geist Mono — loaded via `next/font/google`
- Tailwind custom tokens: `bg-base`, `bg-surface`, `bg-elevated`, `accent`, `text-primary`, `text-secondary`
- All component styling follows the tokens in DESIGN.md — no hardcoded hex values in components
